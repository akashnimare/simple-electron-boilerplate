# Electron desktop app

> Awesome Electron app


## Dev

```
$ npm install
```

### Run

```
$ npm start
```
### Watch

```
$ npm run watch
```

### Build

```
$ npm run build
```

- [x] Basic Menu items

- [x] Restore window on close
- [x] Livereload
- [ ] OSX/Win/Linux installer
- [ ] Launch on OS startup
- [ ] Automatic Updates
